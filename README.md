# C++ GUI Project

## How to Build

### Tools Needed
* CMake (v3.10)

### Steps
Repeat as you develop
1. `mkdir build`
2. `cd build`
3. `cmake ..`
4. `cmake --build .`

## How to Run
`./WCCreate`
